# MorseCode-MominaT
A  code that converts morse code &lt;--> english language
DSA structure use Binary Tree & Hashing
This project implements a Morse Code Converter in C++ that translates normal text to Morse code and Morse code back to text. The program reads Morse patterns from a dictionary text file, builds a Hash Map for fast encoding and a Binary Tree for decoding. It also stores all conversions in a history file for record keeping. The goal was to practice Data Structuresfile handling, and string processing in a real application.
